# 版本补丁

仓库初始化前的完整发布包体积较大，本目录先保存可审阅、可还原的源码差异。

- `0.9.4-to-0.9.5/part-*`：天下史册、公告对齐、外族棋局、人丹对象、宗门整饬与史册名称过滤。
- `0.9.5-to-0.9.6/part-*`：四艺四境分级、固定丹药与符箓效果、阵法速度/维修、炼器稀有度/词条数值、九曜延寿丹消耗金丹金性。

同一目录内的分片按文件名顺序直接拼接即可还原统一 diff，例如：

```bash
cat part-* > version.patch
```
