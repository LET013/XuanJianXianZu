# 0.9.6 世家春秋·四艺四境分级与固定成品体系

## 四艺统一品级

炼丹、炼器、符箓、阵法统一为炼气、筑基、紫府、金丹四个品级。品级决定可处理的最高层次，经验在当前品级内提高对应效率；经验达到阈值且修士境界可承载时，于年度结算自动晋级。

旧六阶迁移：一二阶→炼气级，三四阶→筑基级，五阶→紫府级，六阶→金丹级。

## 固定成品

- 丹药：只保留固定药效、适用境界、基础成丹率和一炉产量；品级与经验提高成丹率和产量。
- 符箓：只保留固定效果、基础成功率和产出率；品级与经验提高成功率和产量。
- 阵法：品级决定工程层次，经验只提高布阵速度和维修效果，不再生成品质或随机劣化。
- 炼器：普通装备只提高稀有度投骰；法器、灵宝、法宝只提高既有上限内的词条数值，不额外提高成功率。
- 九曜延寿丹：九种上品药材各九份，并额外消耗家族重宝仓库中的金丹金性一缕；一炉只成一枚。
